## 📌 Sobre o Projeto

A recolha de informação sobre ciberameaças depende cada vez mais de tecnologias de deceção para capturar e analisar o comportamento dos atacantes. Este projeto documenta o desenho, a implementação e a avaliação de uma infraestrutura híbrida de múltiplos honeypots. 

O grande desafio arquitetónico resolvido neste trabalho foi a adaptação de toda a infraestrutura de monitorização para operar de forma eficaz num ambiente académico e institucional com rigorosos controlos de segurança e restrições de firewall.

## 🚀 Arquitetura e Tecnologias Destacadas

* **Honeypots:** Implementação de Cowrie, Dionaea e serviços de reencaminhamento leves com Socat.
* **Telemetria:** Criação de uma pipeline centralizada assente na Stack ELK (Elasticsearch, Logstash e Kibana).
* **Threat Intelligence:** Classificação automatizada de malware através da integração com a API do VirusTotal.
* **Superfície de Ataque:** Priorização de protocolos-alvo (SSH, HTTP/HTTPS e bases de dados selecionadas) adequados às regras da firewall institucional.
